### ğŸ”§ Installation

1. Clone the repository
```bash
git clone https://github.com/KOSALSENSOK096/learning-managment-system.git
```

2. Navigate to project directory
```bash
cd learning-managment-system
```

3. Install dependencies
```bash
composer install
npm install
```

4. Configure environment
```bash
cp .env.example .env

```

5. Run migrations
