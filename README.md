## 🚀 Quick Start

### Prerequisites

- PHP >= 8.0
- MySQL >= 8.0
- Node.js >= 14.x
- Composer >= 2.0
- Redis >= 6.0 (optional)
- Elasticsearch >= 7.x (optional)

### One-Click Installation

```bash
curl -s https://install.university-lms.com | bash
```

### Manual Installation

1. Clone the repository
```bash
git clone https://github.com/KOSALSENSOK096/learning-managment-system.git
cd learning-managment-system
```

2. Install dependencies
```bash
composer install
npm install
```

3. Configure environment
```bash
cp .env.example .env
php artisan key:generate
```

4. Run migrations and seeders
```bash
php artisan migrate --seed
```

5. Build assets and start servers
```bash
npm run build
php artisan serve
```

## 🛠️ Tech Stack

### 🎨 Frontend
- Vue.js 2.6 with Composition API
- TailwindCSS 3.0 for styling
- Material Design components
- Progressive Web App support
- Real-time updates with WebSocket
- Responsive design system
- State management with Vuex
- Route management with Vue Router

### ⚙️ Backend
- Laravel 8.x framework
- MySQL 8.0 database
- Redis for caching
- Elasticsearch for search
- WebSocket for real-time features
- REST API architecture
- JWT authentication
- Rate limiting & security

### 🚀 DevOps
- Docker containerization
- GitHub Actions CI/CD
- AWS Cloud infrastructure
- New Relic monitoring
- Automated testing
- Performance optimization
- Security scanning
- Automated backups
